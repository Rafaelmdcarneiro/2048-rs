# rust-2048

[![Build Status](https://travis-ci.org/andelf/rust-2048.svg?branch=master)](https://travis-ci.org/andelf/rust-2048)

2048 game in rust. Original is in [2048](http://gabrielecirulli.github.io/2048/).

Working in Progress.

## How to Install/Play

Refer ``.travis.yml``. :)

In short!

    cargo build
